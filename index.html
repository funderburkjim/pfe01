Polymer First Element Tutorial

This is a minor variant of the Polymer project tutorial (version 1.0) described here. I'm not sure how this differs from the Polymer2.0 example.

The links below show the results at various stages of the tutorial.

The main difference from the referenced tutorial is that there are slight changes to the paths mentioned in <link> and <script> tags so that normal server conventions apply, rather than the odd conventions used in the polymer server that is part of the nodejs polymer-cli system. These changes appear only in the html files modified in the tutotials that appear in directories step2, step3, etc. The underlying files of the components that appear in subfolders of the bower_components directory are unchanged and copied directly from the tutorial download.

One other difference is that the webcomponentsjs and polymer directories in bower_components come from a different source. I think they are from a version of version 2 of the tutorial. I got them from the cold-umbrella example mentioned here.

Another difference regards a detail of naming in the various directories (step1, step2, etc.) The original naming of the files in the tutorial is:
index.html, icon-toggle.html, icon-toggle-demo.html.
These are changed, in the step1 directory to
index.html (same), icon-toggle1.html, icon-toggle1-demo.html.
Also, within the files, the component names are similarly changed to:
icon-toggle1-demo and icon-toggle1.
